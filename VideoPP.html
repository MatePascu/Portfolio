En esta pasantía que realicé para Social Learning utilize el framework Vue.js para crear el portal, con varios pluggins como Vuex, para guardar datos en estados, y Vue Router, para realizar en enrutamiento entre módulos y realizar validaciones de seguridad. Usamos componentes de Vuetify y para conectarnos con los servicios creados por el BackEnd utilizamos promesas de Axios. Tambien hicimos uso de las cookies para guardar la autorización de ingreso al portal y algunos datos del usuario.
